## Overview

**morpho-vault-rwa-connector** is an open-source integration for Real World Asset tokenization and on-chain finance.

Solidity adapters to integrate RWA collateral into Morpho Blue vaults.

Topics: `lending` `morpho` `defi`

## Features

- 🚀 Production-ready integration targeting rwa use cases
- 🔧 Easy to integrate into existing projects
- 📦 Zero-configuration defaults with full customization support
- 🧪 Comprehensive test coverage
- 📖 Well-documented API with examples
